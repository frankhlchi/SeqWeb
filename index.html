Unifying and Optimizing Data Values for Selection via Sequential Decision-Making

Abstract

Data selection has emerged as a crucial downstream application of data valuation, yet the theoretical foundations for using data values in selection remain underexplored. We reformulate data selection as a sequential decision-making problem where the optimal selection sequence arises from dynamic programming, and data values can be understood as encodings of this optimal sequence. This framework unifies and reinterprets existing methods like Data Shapley through the lens of approximate dynamic programming, revealing them as myopic linear approximations to the sequential problem. We further analyze how selection optimality degrades with utility curvature under submodularity, explaining when and why these approximations fail. To bridge theory and practice, we propose an efficient bipartite graph-based surrogate that preserves submodular structure while enabling scalable greedy selection with provable guarantees. Experiments on classical ML benchmarks and large-scale LLM fine-tuning data selection demonstrate substantial improvements over existing methods.

Overview

We often want to keep only the most useful subset of training data. Data-valuation methods such as Data Shapley assign each example a score, and we then select the top-scoring ones. But why score-then-select works — and when it breaks — has been poorly understood. Our key move is to treat selection not as one-shot scoring, but as a sequence of decisions whose optimum is given by dynamic programming. A “data value” is then best understood as an encoding of that optimal selection sequence.

Through this lens, classical methods like Data Shapley turn out to be myopic, one-step linear approximations of the true sequential problem. Using submodularity and utility curvature, we characterize when those linear approximations degrade — and replace them with a submodularity-preserving bipartite-graph surrogate that scales to large data and comes with provable guarantees.

Results

We evaluate the proposed bipartite-based method against standard data-valuation baselines for selection on eight datasets, and validate the curvature analysis on controlled data. See the paper for full results, including large-scale LLM fine-tuning data selection.

Key Contributions

  • A sequential view of selection. We cast data selection as a dynamic program whose optimal selection sequence defines what a “data value” should encode.
  • A unifying theory. Data Shapley and related semi-values emerge as myopic linear approximations to this sequential problem under approximate dynamic programming.
  • When approximations fail. Under submodularity, we quantify how selection optimality degrades with utility curvature — explaining when and why existing methods break down.
  • A scalable, provable surrogate. An efficient bipartite graph-based surrogate preserves submodular structure and enables greedy selection with provable guarantees.
  • Strong empirical results. Substantial improvements over prior methods on classical ML benchmarks and large-scale LLM fine-tuning data selection.
